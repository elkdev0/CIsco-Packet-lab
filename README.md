# CIsco-Packet-lab

<img width="1680" height="833" alt="image" src="https://github.com/user-attachments/assets/30989888-f63b-4246-a0cc-fb45215ba789" />

This project demonstrates the design and implementation of a home cloud network-attached storage (NAS) system using a Raspberry Pi.

I first built and validated the network architecture in Cisco Packet Tracer, then mapped the same configuration to physical hardware. The project highlights network segmentation, secure remote access, and practical home-lab deployment.
